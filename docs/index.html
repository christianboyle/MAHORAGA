How it works

Monitor. Analyze. Execute.

The agent monitors StockTwits and Reddit for trending tickers. Each signal is researched by the LLM — analyzing sentiment, timing, catalysts, and red flags. If confidence is high enough, it executes via Alpaca and monitors the position.

FAQ

Common questions

What LLM providers are supported?
OpenAI, Anthropic, Google, xAI, and DeepSeek via Vercel AI SDK. Or use Cloudflare AI Gateway for unified billing.
How much does it cost to run?
LLM costs are typically $0.50-2/day using gpt-4o-mini. Cloudflare Workers free tier. Alpaca is free.
Is this safe to use with real money?
Paper trading is enabled by default. Built-in safety includes kill switch, stop losses, position limits, and daily loss limits.
How does sentiment analysis work?
Monitors StockTwits and Reddit for trending tickers. Each signal is researched by the LLM before returning a confidence score.
Can I customize the strategy?
Yes. Create a strategy in src/strategy/my-strategy/ and change one import line. Reuse default gatherers, override prompts, define your own entry/exit rules.
What if something goes wrong?
Emergency kill switch disables trading immediately. Stop losses protect positions. Daily loss limit halts after 2% drawdown.
